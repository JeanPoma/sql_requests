## Ressources SQL

### Documentation Officielle

- [Documentation MariaDB](https://mariadb.com/kb/en/documentation/): La ressource officielle pour les concepts, la
  syntaxe et les fonctionnalités spécifiques à MariaDB.

### Tutoriels SQL

1. **Introduction au SQL avec MariaDB**
   - [Getting Started with MariaDB](https://mariadb.com/kb/en/getting-started-with-sql/) : Idéal pour apprendre les
     bases du SQL et pratiquer avec MariaDB.

2. **Requêtes de Sélection**
   - [MariaDB SELECT Statement](https://mariadb.com/kb/en/select/) : Apprenez la syntaxe et les options disponibles pour
     les requêtes de sélection.

3. **Requêtes avec Fonctions d'Agrégation**
   - [Aggregate functions in MariaDB](https://mariadb.com/kb/en/aggregate-functions/) : Détails sur les fonctions comme
     `SUM()`, `AVG()`, `COUNT()`, etc.

4. **Jointures**
   - [Understanding Joins in MariaDB](https://mariadb.com/kb/en/joins-overview/) : Explication détaillée sur les
     jointures (INNER JOIN, LEFT JOIN, RIGHT JOIN, etc.) avec exemples.

### Pratique en Ligne

- [SQL Fiddle](http://sqlfiddle.com/) : Testez vos requêtes SQL sur un environnement en ligne compatible avec MariaDB.
- [DB Fiddle](https://www.db-fiddle.com/) : Autre alternative pour expérimenter des requêtes directement dans MariaDB.

### Blogs/Tutoriels Complémentaires

- [Basic SQL Commands for MariaDB](https://database.guide/basic-sql-commands-for-mariadb/) : Une introduction rapide
  avec des exemples concrets.
- [MariaDB SQL Optimization](https://mariadb.com/kb/en/query-optimizations/) : Guide pour optimiser vos requêtes dans
  MariaDB.

### Exercices et Plateformes d'Apprentissage

- [SQLZoo: MariaDB Exercises](https://sqlzoo.net/wiki/MariaDB_Tutorial) : Plateforme interactive spécialisée, parfaite
  pour pratiquer les concepts basiques comme avancés.
